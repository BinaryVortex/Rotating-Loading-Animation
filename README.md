# Rotating Loading Animation

A clean, lightweight rotating loading animation built using plain HTML, CSS and a touch of JavaScript.

## Preview

![Rotating Loading Animation](https://raw.githubusercontent.com/BinaryVortex/Rotating-Loading-Animation/main/Screenshot%202024-05-23%20223312.png)

## About

This small demo shows a simple rotating loader that you can drop into any web project. It's intentionally minimal so you can easily customize size, color, and speed.

## Features

- Pure HTML & CSS (small JS for demo if needed)
- Easy to customize (colors, size, animation speed)
- Lightweight and responsive

## Files

- `index.html` — markup for the loader demo
- `style.css` — all styles and animation rules

## Usage

1. Clone or download the repository.
2. Open `index.html` in your browser to view the animation locally.

Or run a simple local server:

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000
```

## Customization

- To change the color or size, edit `style.css`.
- To change the rotation speed, look for the `animation`/`@keyframes` block in `style.css` and adjust the duration.
